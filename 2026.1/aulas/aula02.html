O que são Operadores?

Operadores são símbolos especiais que dizem ao Python para realizar uma operação.

Aritméticos
Fazem contas
+ - * /
Relacionais
Comparam valores
> < == !=
Lógicos
Combinam condições
and or not

Operadores Aritméticos

Operação Símbolo Exemplo Resultado
Soma + 5 + 3 8
Subtração - 10 - 4 6
Multiplicação * 3 * 7 21
Divisão / 10 / 3 3.333…
Divisão inteira // 10 // 3 3
Resto (módulo) % 10 % 3 1
Potência ** 2 ** 3 8

Aritméticos na Prática


a = 15
b = 4

print(a + b)   # 19   → soma
print(a - b)   # 11   → subtração
print(a * b)   # 60   → multiplicação
print(a / b)   # 3.75 → divisão
print(a // b)  # 3    → divisão inteira
print(a % b)   # 3    → resto
print(a ** b)  # 50625 → potência (15⁴)
/ sempre retorna float. Use // quando quiser só a parte inteira.

print(7 / 2)   # 3.5

print(1 / 3)   # 0.333…

Sempre retorna float

// Divisão inteira


print(7 // 2)   # 3
print(10 // 5)  # 2
print(1 // 3)   # 0

Descarta a parte decimal

O Operador % (Módulo / Resto)

Retorna o resto da divisão inteira.


print(10 % 3)  # 1  → 10 ÷ 3 = 3, resto 1
print(10 % 2)  # 0  → 10 ÷ 2 = 5, resto 0
print(7 % 4)   # 3  → 7 ÷ 4 = 1, resto 3

Exemplo: Calculadora Simples


n1 = float(input("Digite o 1º número: "))

Cuidado! Dividir por zero causa erro: ZeroDivisionError.

Precedência Aritmética

Quando uma expressão tem vários operadores, o Python resolve em uma ordem padrão.

1) () parênteses 2) ** potência 3) * / // % 4) + - 

resultado1 = 2 + 3 * 4      # 14
resultado2 = (2 + 3) * 4    # 20

Resolva (use a precedência: parênteses → potência → mult/div/mod → soma/sub).

  1. 1

    2 + 3 * 4;

  2. 2

    18 / 3 + 5 * 2;

  3. 3

    (9 - 3) * (2 + 1);

  4. 4

    20 // 3 + 20 % 3;

  5. 5

    2 ** 3 + 4 * 2 - 5;

Atividade curta: Resolva primeiro sem executar e depois confira no Python.

Respostas ✓

  1. 1

    2 + 3 * 4;

    = 14

  2. 2

    18 / 3 + 5 * 2;

    = 16

  3. 3

    (9 - 3) * (2 + 1);

    = 18

  4. 4

    20 // 3 + 20 % 3;

    = 8

  5. 5

    2 ** 3 + 4 * 2 - 5;

    = 11

Operadores Relacionais

Servem para comparar dois valores.

O resultado é sempre True ou False (tipo bool).

São essenciais para criar condições (if) nas próximas aulas!

Tabela de Operadores Relacionais

Operador Significado Exemplo Resultado
== Igual a 5 == 5 True
!= Diferente de 5 != 3 True
> Maior que 7 > 3 True
< Menor que 2 < 8 True
>= Maior ou igual 5 >= 5 True
<= Menor ou igual 3 <= 2 False

Relacionais na Prática


a = 10
b = 7

print(a == b)   # False → 10 não é igual a 7
print(a != b)   # True  → 10 é diferente de 7
print(a > b)    # True  → 10 é maior que 7
print(a < b)    # False → 10 não é menor que 7
print(a >= 10)  # True  → 10 é maior ou igual a 10
print(b <= 5)   # False → 7 não é menor ou igual a 5

Cuidado! = vs ==

= (um igual)
Atribuição — guarda valor

x = 10  # x recebe 10
== (dois iguais)
Comparação — verifica igualdade

x == 10  # x é igual a 10?

Esse é um dos erros mais comuns de iniciantes!

Exemplo com Entrada do Usuário


idade = int(input("Qual a sua idade? "))

print(f"Você é maior de idade? {idade >= 18}")
print(f"Você tem exatamente 20 anos? {idade == 20}")
print(f"Você tem menos de 16 anos? {idade < 16}")
Cada expressão retorna True ou False automaticamente.

Operadores Lógicos

Servem para combinar duas ou mais condições.

Resultado: sempre True ou False.

Operador Significado Quando é True?
and E Ambas as condições são verdadeiras
or OU Pelo menos uma condição é verdadeira
not NÃO Inverte o valor lógico

Tabela Verdade — and

Só é True quando as duas condições são verdadeiras.

A B A and B
True True True
True False False
False True False
False False False

Tabela Verdade — or

É True quando pelo menos uma condição é verdadeira.

A B A or B
True True True
True False True
False True True
False False False

Tabela Verdade — not

Inverte o valor lógico.

A not A
True False
False True 

chovendo = True
print(not chovendo)  # False → "não está chovendo" é falso

Exemplo do Dia a Dia

Imagine um sistema de cinema que dá desconto para estudantes menores de 25 anos:


estudante = input("Você é estudante? (s/n): ")
idade = int(input("Qual a sua idade? "))

tem_desconto = (estudante == "s") and (idade < 25)

print(f"Tem desconto? {tem_desconto}")
Usar parênteses deixa o código mais legível, mesmo quando não são obrigatórios.

Combinando Operadores

Você pode misturar aritméticos, relacionais e lógicos!


nota1 = 7.0
nota2 = 5.0
media = (nota1 + nota2) / 2          # aritmético
faltas = 3

aprovado = (media >= 6) and (faltas <= 5)  # relacional + lógico

print(f"Média: {media}")
print(f"Aprovado? {aprovado}")  # True

Ordem de Precedência

Python segue uma ordem para avaliar os operadores (do mais prioritário ao menos):

Prioridade Operadores Tipo
1ª (mais alta) ** Potência
2ª * / // % Multiplicação, divisão
3ª + - Soma, subtração
4ª == != > < >= <= Relacionais
5ª not Lógico NÃO
6ª and Lógico E
7ª (mais baixa) or Lógico OU

Na dúvida, use parênteses!
