Conceito

F-Strings em Python

A forma moderna e elegante de formatar texto com variáveis.

O que é uma F-String?

Uma f-string é uma string especial que permite inserir variáveis e expressões diretamente dentro do texto.


valor = 10
print(f"Olá, {valor}!")
              

nome = "Ana"
print(f"Olá, {nome}!")

Sintaxe da F-String

O f antes das aspas "ativa" a interpolação. As chaves { } são avaliadas em tempo real:


nome  = "Carlos"
idade = 22
curso = "Python"

# Variáveis dentro das chaves
print(f"Nome: {nome}")
print(f"Idade: {idade} anos")
print(f"{nome} tem {idade} anos e estuda {curso}.")

# Saída:
# Nome: Carlos
# Idade: 22 anos
# Carlos tem 22 anos e estuda Python.

Formatando Números

Após o valor, use : seguido de um especificador de formato:


preco  = 1234.5678
pi     = 3.14159265
taxa   = 0.1525
numero = 42

print(f"2 casas decimais:  R$ {preco:.2f}")   # R$ 1234.57
print(f"4 casas decimais:  {pi:.4f}")         # 3.1416
print(f"Percentual:        {taxa:.1%}")       # 15.2%
print(f"Inteiro sem fração:{preco:.0f}")      # 1235

Atividade 1 de 5

Média de Três Notas

Crie três variáveis: nota1, nota2 e nota3.

Calcule a média aritmética das três notas e armazene em uma variável chamada media.

Exiba o resultado com a mensagem: "A média é: X.XX"

Dica: use print(f"A média é: {media:.2f}") para formatar com 2 casas decimais.

Resposta — Atividade 1

Solução


nota1 = 8.0
nota2 = 6.5
nota3 = 9.0

media = (nota1 + nota2 + nota3) / 3

print(f"A média é: {media:.2f}")
# Saída: A média é: 7.83
Pontos-chave: os parênteses garantem que a soma ocorra antes da divisão, e :.2f formata com 2 casas decimais.

Atividade 2 de 5

Conversor de Temperatura

Leia uma temperatura em Celsius com input().

Converta para Fahrenheit usando a fórmula:

F = C × 1.8 + 32

Exiba: "X C equivale a Y F"

Dica: converta a entrada para float antes de calcular.

Resposta — Atividade 2

Solução


celsius = float(input("Digite a temperatura em °C: "))

fahrenheit = celsius * 1.8 + 32

print(f"{celsius}C equivale a {fahrenheit:.1f}F")
# Exemplo: 100C equivale a 212.0F
Pontos-chave: float(input()) converte a string para número decimal. A fórmula deve ser aplicada antes do print.

Atividade 4 de 5

Área e Perímetro do Retângulo

Leia a largura e a altura de um retângulo.

Calcule e exiba:

  • Área = largura × altura
  • Perímetro = 2 × (largura + altura)
  • Se a área é maior que 50: exiba True ou False

Dica: use ** e * para calcular. Mostre os resultados com f-string.

Resposta — Atividade 4

Solução


largura = float(input("Largura: "))
altura  = float(input("Altura: "))

area       = largura * altura
perimetro  = 2 * (largura + altura)
area_grande = area > 50

print(f"Área: {area:.2f}")
print(f"Perímetro: {perimetro:.2f}")
print(f"Área maior que 50? {area_grande}")
Pontos-chave: a expressão area > 50 já retorna um booleano (True/False) que pode ser armazenado em variável e exibido normalmente.

Atividade 5 de 5

Salário com Bônus

Leia o salário base e o percentual de bônus de um funcionário.

Calcule e exiba:

  • O valor do bônus = salário × (percentual / 100)
  • O salário final = salário + bônus
  • Se o salário final supera R$ 5.000: True ou False

Dica: exiba todos os valores com 2 casas decimais usando :.2f.

Resposta — Atividade 5

Solução


salario    = float(input("Salário base: R$ "))
percentual = float(input("Percentual de bônus (%): "))

bonus        = salario * (percentual / 100)
salario_final = salario + bonus
acima_5000   = salario_final > 5000

print(f"Bônus:         R$ {bonus:.2f}")
print(f"Salário final: R$ {salario_final:.2f}")
print(f"Acima de R$5.000? {acima_5000}")
Pontos-chave: dividir por 100 converte o percentual para fator decimal. As expressões aritméticas e relacionais podem ser combinadas em sequência de forma natural.
