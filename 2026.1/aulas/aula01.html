O que é Programar?

Programar é dar instruções para o computador executar.

É como escrever uma receita: cada passo deve ser claro.

O computador é muito rápido, mas não "pensa" sozinho.
Ele faz exatamente o que você manda.

Algoritmo no Dia a Dia

Algoritmo = sequência de passos para resolver um problema.

Fazer um café

  1. Ferver a água
  2. Colocar o pó no filtro
  3. Despejar a água
  4. Servir na xícara

Escovar os dentes

  1. Pegar a escova
  2. Colocar pasta
  3. Escovar por 2 min
  4. Enxaguar a boca

Você já usa algoritmos todos os dias!

Características de um Algoritmo

  • Claro – cada passo é compreensível
  • Ordenado – a sequência importa
  • Finito – tem começo, meio e fim
  • Eficaz – resolve o problema proposto

Do Algoritmo ao Código

Algoritmo


1. Perguntar o nome
2. Saudar a pessoa

Python


nome = input("Qual é o seu nome? ")
print("Olá,", nome, "!")

O que é uma Variável?

Pense numa variável como uma caixa com etiqueta.

Você dá um nome para a caixa e guarda um valor dentro.


nome = "Ana"
idade = 19
O sinal = em Python não é "igual" da matemática.
Ele significa "recebe" → nome recebe "Ana".

Regras para Nomes de Variáveis

Válido Inválido Motivo
nome 1nome Não pode começar com número
nota_final nota final Não pode ter espaço
idade2 class Palavra reservada do Python

Tipos de Dados em Python

Tipo O que guarda Exemplo
int Números inteiros 10, -3, 0
float Números com decimal 3.14, -0.5
str Textos (strings) "Olá", 'Python'
bool Verdadeiro / Falso True, False

Tipos na Prática


nome = "Maria"           # str  → texto
idade = 20               # int  → inteiro
altura = 1.65            # float → decimal
matriculada = True       # bool → verdadeiro

print(type(nome))        # <class 'str'>
print(type(idade))       # <class 'int'>
print(type(altura))      # <class 'float'>
print(type(matriculada)) # <class 'bool'>
A função type() mostra o tipo de qualquer variável.
