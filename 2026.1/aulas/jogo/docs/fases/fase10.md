# Fase 10 — Decisão com If

## Objetivo
Resolver um evento aleatório simples usando `if/else` e chegar à bandeira.

## Comandos
- `move_right(n)`
- `move_left(n)`
- `interact()`
- `jump()`
- `if is_crystal_ahead():`
- `else:`

## Mecânica
- Um evento aparece na rota: cristal ou obstáculo.
- Se for cristal, use `interact()`.
- Se for obstáculo, use `jump()`.
- Depois de resolver o evento, avance até a bandeira.

## Dica
Teste o mesmo código várias vezes: o evento é aleatório e o `if/else` precisa funcionar nos dois casos.
