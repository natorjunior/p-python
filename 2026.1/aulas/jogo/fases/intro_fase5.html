Passo 1: Calculo de Rota

Agora voce vai usar variaveis para calcular distancias e montar uma rota completa com ida e volta.

Passo 2: Novo desafio

Nao basta chegar na bandeira. Primeiro voce precisa coletar os cristais A e B.

Passo 3: Variavel base

base = 2
Comece com um valor simples e reaproveite esse valor nos calculos.

Passo 4: Somando distancias

rota1 = base + 3
Use + para aumentar a distancia.

Passo 5: Subtraindo distancias

rota2 = rota1 - 2
Use - para ajustar o retorno.

Passo 6: Variavel calcula variavel

rota3 = rota1 + 1
Uma variavel pode usar o resultado de outra variavel.

Passo 7: Movimento com variaveis

move_right(rota1)
O comando de movimento usa o valor guardado na variavel.

Passo 8: Ida e volta

Nesta fase, voce precisa usar move_right() e move_left(). Isso deixa a rota mais estrategica.

Passo 9: Exemplo de sequencia

base = 2
rota1 = base + 3
rota2 = rota1 - 2
rota3 = rota1 + 1

Passo 10: Execute a rota

move_right(rota1)
move_left(rota2)
move_right(rota3)

Passo 11: Erros comuns

1) Nao usar soma e subtracao nas atribuicoes.
2) Criar rota que nao passa pelos cristais A e B.
3) Tentar ir para a bandeira sem coletar A e B.
