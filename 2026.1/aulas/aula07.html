Parte 1

Laço while

Repete um bloco enquanto uma condição for verdadeira.

Sintaxe do while


while condicao:
    # bloco de repetição
    # atualização da variável de iteração

Leitura: "enquanto a condição for verdadeira, execute este bloco."

Ponto-chave: a variável de iteração precisa mudar a cada repetição.

Fluxograma 1

Condicional simples (if)

n = 0, então n > 0 é falsa desde o início.

O bloco do if não executa.

O programa pula direto para print('Secar!').

Secar!

Fluxograma 1

Condicional simples (if) — Código

Código do fluxograma


n = 0
if n > 0:
    print("Ensaboar")
    print("Enxaguar")
print("Secar!")

Fluxograma 2

loop

Agora n é decrementado a cada volta.

O laço roda enquanto n > 0: valores 5, 4, 3, 2, 1.

Quando n = 0, a condição falha — o laço termina.

5,4,3,2,1 Fim!

Fluxograma 2

Loop controlado — Código

Código do fluxograma


n = 5
while n > 0:
    print(n)
    n = n - 1
print("Fim!")

Fluxograma 3

loop

n = 5, então n > 0 é verdadeira.

O laço executa... mas n nunca muda!

→ Laço infinito. O programa trava.

Ensaboar Enxaguar Ensaboar Enxaguar ... (nunca termina)

Fluxograma 3

Laço infinito — Código

Código do fluxograma


n = 5
while n > 0:
    print("Ensaboar")
    print("Enxaguar")
print("Secar!")

Atividade 1 — while

Contagem de 1 até N

Leia n e mostre os números de 1 até n usando while.

Exemplo: se n = 5, a saída deve ser 1 2 3 4 5.

Pseudocódigo:


ler n
contador ← 1

enquanto contador ≤ n faça
    imprimir contador
    contador ← contador + 1
fim_enquanto

Comece em 1 e aumente de 1 em 1 até chegar em n.

Resposta — Atividade 1

Solução com while


n = int(input("Digite n: "))
contador = 1

while contador <= n:
    print(contador)
    contador += 1
Fluxo: inicializar → testar condição → mostrar valor → atualizar → repetir.

Atividade 2 — while

Contagem regressiva

Leia n e exiba uma contagem regressiva de n até 1 usando while.

Exemplo: se n = 4, saída: 4 3 2 1.

Pseudocódigo:


ler n
enquanto n ≥ 1 faça
    imprimir n
    n ← n - 1
fim_enquanto

Resposta — Atividade 2

Solução com while


n = int(input("Digite n: "))

while n >= 1:
    print(n)
    n -= 1
Mesma lógica do laço: testa condição, executa, atualiza e repete.

Atividade 3 — while

Soma até digitar 0

Leia números inteiros e some todos até o usuário digitar 0.

No final, mostre a soma total.

Pseudocódigo:


soma ← 0
ler numero
enquanto numero ≠ 0 faça
    soma ← soma + numero
    ler numero
fim_enquanto
imprimir soma

Resposta — Atividade 3

Solução com while


soma = 0
numero = int(input("Digite um número (0 para sair): "))

while numero != 0:
    soma += numero
    numero = int(input("Digite um número (0 para sair): "))

print(f"Soma total: {soma}")
Aqui usamos 0 como valor sentinela para encerrar o laço.
Parte 2

Laço for

Mais indicado quando a quantidade de repetições é conhecida.

Sintaxe do for com range()


for variavel in range(inicio, fim, passo):
    # bloco de repetição
  • inicio: opcional, padrão 0.
  • fim: limite superior (não incluído).
  • passo: opcional, padrão 1.

Exemplos de range()


for i in range(5):
    print(i)        # 0, 1, 2, 3, 4

for i in range(1, 6):
    print(i)        # 1, 2, 3, 4, 5

for i in range(10, 0, -2):
    print(i)        # 10, 8, 6, 4, 2

Exemplo: percorrendo caracteres


nome = "python"
for letra in nome:
    print(letra)

Atividade 4 — for

Contagem de 1 até 10 com for

Mostre os números de 1 até 10, um por linha, usando for.

Pseudocódigo:


para numero de 1 até 10 faça
    escrever numero
fim_para

Dica: use range(1, 11).

Resposta — Atividade 4

Solução com for


for numero in range(1, 11):
    print(numero)
Quando a repetição é fixa, o for é a forma mais simples de resolver.
Parte 3

break e continue

Controle fino de fluxo dentro dos laços.

break: encerra o laço


while True:
    linha = input("> ")
    if linha == "fim":
        break
    print(linha)
print("Agora é o fim!")
break interrompe o laço corrente e segue para a próxima instrução após ele.

continue: pula a iteração atual


while True:
    linha = input("> ")
    if linha.startswith("#"):
        continue
    if linha == "fim":
        break
    print(linha)
print("Agora é o fim!")
continue encerra só a iteração atual e retorna ao teste da condição.

Escolha da estrutura

Cenário Estrutura indicada Exemplo
Quantidade desconhecida de repetições while Validar senha
Quantidade fixa de repetições for Tabuada de 1 a 10
Interromper o laço imediatamente break Sair ao digitar "fim"
Ignorar apenas uma iteração continue Ignorar linhas de comentário

Desafio rápido (em dupla)

Contador de linhas válidas

Crie um programa que:

1. Leia entradas em loop com while True.

2. Ignore linhas que começam com # usando continue.

3. Finalize ao receber fim com break.

4. Mostre no final quantas linhas válidas foram processadas.

Resumo da Aula

while

Repete enquanto a condição for verdadeira.

for

Percorre sequências com sintaxe enxuta.

break

Interrompe o laço imediatamente.

continue

Pula apenas a iteração atual.
