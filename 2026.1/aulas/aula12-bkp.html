Parte 1

Break e Continue

Comandos para controlar o comportamento do loop no meio da execucao.

O que faz break?


for n in range(1, 11):
    if n == 6:
        break
    print(n)

# saida: 1 2 3 4 5

break encerra o loop imediatamente, sem continuar as proximas iteracoes.

O que faz continue?


for n in range(1, 8):
    if n == 4:
        continue
    print(n)

# saida: 1 2 3 5 6 7

continue pula apenas a iteracao atual e segue o loop normalmente.

Comparacao Rapida

Comando Efeito Uso tipico
break Interrompe o loop Parar busca quando encontrou o alvo
continue Pula para proxima iteracao Ignorar item invalido e continuar

Atividade B1 — Busca com Break

Parada antecipada

Leia numeros inteiros em loop.

Quando o usuario digitar 0, interrompa o loop com break.

Mostre quantos numeros positivos foram digitados antes do zero.

Tempo sugerido: 06:00

Solucao — Atividade B1

Codigo


qtd_positivos = 0

while True:
    n = int(input("Digite um numero (0 para parar): "))
    if n == 0:
        break
    if n > 0:
        qtd_positivos += 1

print("Positivos digitados:", qtd_positivos)

Atividade B2 — Filtro com Continue

Ignorando valores

Percorra os numeros de 1 ate 20.

Use continue para ignorar os multiplos de 3.

Imprima apenas os valores restantes.

Tempo sugerido: 05:00

Solucao — Atividade B2

Codigo


for n in range(1, 21):
    if n % 3 == 0:
        continue
    print(n)
O loop continua, mas os valores que atendem a condicao sao pulados.
Parte 2

Tuplas

Estrutura ordenada e imutavel para representar pares e registros fixos.

Criacao e Acesso


coordenada = (10, 25)
print(coordenada[0])  # 10
print(coordenada[1])  # 25

aluno = ("Ana", 8.7)
nome = aluno[0]
nota = aluno[1]
  • Tuplas usam parenteses: ().
  • Podem guardar tipos diferentes no mesmo registro.

Imutabilidade


cores = ("azul", "verde", "amarelo")

# cores[1] = "vermelho"  -> erro!
print(cores)

Depois de criada, a tupla nao permite alteracao direta dos elementos.

Percorrendo Tuplas


produtos = [("Arroz", 12.5), ("Feijao", 9.8), ("Oleo", 7.2)]

for nome, preco in produtos:
    print(nome, "-> R$", preco)

Esse padrao de desempacotamento em for nome, valor in ... e muito comum.

Atividade T1 — Tupla de Produtos

Lista de compras

Crie uma lista com 4 tuplas no formato (produto, preco).

Percorra a lista e mostre cada produto com seu preco.

No final, mostre a soma total dos precos.

Tempo sugerido: 07:00

Solucao — Atividade T1

Codigo


itens = [("Arroz", 12.5), ("Feijao", 9.8), ("Oleo", 7.2), ("Cafe", 15.0)]
total = 0

for produto, preco in itens:
    print(produto, "-> R$", preco)
    total += preco

print("Total:", total)
