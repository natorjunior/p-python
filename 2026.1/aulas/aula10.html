Atividade 4 — Vetores PAR e IMPAR

Separacao de numeros

Leia 20 numeros inteiros e armazene no vetor numeros.

Armazene os pares no vetor par e os impares no vetor impar.

Imprima os tres vetores no final.

Tempo sugerido: 07:00

Solucao — Atividade 4

Codigo


numeros = []
par = []
impar = []

for i in range(20):
    n = int(input(f"Digite o {i+1}o numero: "))
    numeros.append(n)
    if n % 2 == 0:
        par.append(n)
    else:
        impar.append(n)

print("Numeros:", numeros)
print("PAR:", par)
print("IMPAR:", impar)

Atividade D1 — Dicionarios

Controle de estoque

Considere o dicionario:

estoque = {"arroz": 12, "feijao": 4, "macarrao": 7, "oleo": 3}

Percorra o dicionario e informe:

1) Quantidade total de itens em estoque.

2) Quais produtos estao com estoque baixo (< 5).

Dica: use baixo_estoque = [] e adicione produtos com append().

Resposta — Atividade D1

Solucao com Dicionario


estoque = {"arroz": 12, "feijao": 4, "macarrao": 7, "oleo": 3}

total_itens = 0
baixo_estoque = []

for produto, qtd in estoque.items():
    total_itens += qtd
    if qtd < 5:
        baixo_estoque.append(produto)

print("Total de itens:", total_itens)              # 26
print("Produtos com estoque baixo:", baixo_estoque) # ['feijao', 'oleo']
for chave, valor in dicionario.items() e o padrao padrao para dicionarios. Combinar com lista de coleta (append) e muito util.

Atividade D2 — Dicionarios com if/elif/else

Situacao dos alunos

Considere:

notas = {"Ana": 8.5, "Bruno": 5.2, "Carla": 6.8, "Diego": 4.5}

Classifique cada aluno em tres niveis:

    ≥ 7.0 → aprovado

    5.0 a 6.9 → recuperacao

    < 5.0 → reprovado

Ao final, exiba o total de aprovados e reprovados.

Resposta — Atividade D2

Solucao com if/elif/else


notas = {"Ana": 8.5, "Bruno": 5.2, "Carla": 6.8, "Diego": 4.5}
aprovados = 0
reprovados = 0

for nome, nota in notas.items():
    if nota >= 7:
        situacao = "aprovado"
        aprovados += 1
    elif nota >= 5:
        situacao = "recuperacao"
    else:
        situacao = "reprovado"
        reprovados += 1
    print(nome, "->", situacao)

print("Aprovados:", aprovados)   # 1
print("Reprovados:", reprovados) # 1
if/elif/else dentro do loop classifica em tres categorias; variaveis de contagem separadas rastreiam cada grupo de interesse.
