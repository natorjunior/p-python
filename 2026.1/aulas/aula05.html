  • Usamos operadores aritméticos para calcular valores.

Com isso, já conseguimos montar condições. Agora vamos decidir qual caminho o programa deve seguir!

O Que é Condicional?

Na vida real, tomamos decisões o tempo todo:

🌧️
Se estiver chovendo, levo o guarda-chuva.
Senão, saio sem ele.
📝
Se a nota for ≥ 7, o aluno está aprovado.
Senão, está reprovado.
Em Python, fazemos isso com if, elif e else.

Fluxo de Decisão

O programa avalia uma condição e escolhe um caminho:


Condição
→
✅ Verdadeiro
executa bloco A
❌ Falso
executa bloco B
Em Python, o bloco de código é definido pela indentação (4 espaços).
Parte 1

Estrutura if

Executando código apenas quando a condição é verdadeira.

Estrutura if

O if executa um bloco somente se a condição for verdadeira:


idade = int(input("Digite sua idade: "))

if idade >= 18:
    print("Maior de idade")
Se a condição for falsa, nada acontece — o programa simplesmente continua.

Micro Fixação

O que será exibido?


x = 10
if x > 20:
    print("Grande")
print("Fim")
a) Grande e Fim
b) Apenas Fim
c) Apenas Grande
d) Nada é exibido
Resposta: b) — x > 20 é falso, então o print("Grande") não roda. Mas print("Fim") está fora do if.
Parte 2

Estrutura if / else

Dois caminhos: um para verdadeiro, outro para falso.

Estrutura if / else

Quando queremos que algo aconteça nos dois casos:


nota = float(input("Digite a nota: "))

if nota >= 7:
    print("Aprovado ✅")
else:
    print("Reprovado ❌")
O else pega todos os outros casos que o if não pegou.

Micro Fixação

Qual a saída para cada valor?


n = int(input("Número: "))
if n % 2 == 0:
    print("Par")
else:
    print("Ímpar")
n = 4 → Saída: ?
n = 7 → Saída: ?
n = 0 → Saída: ?
Respostas: Par, Ímpar, Par — zero é par porque 0 % 2 == 0 é verdadeiro.
Parte 3

Estrutura if / elif / else

Múltiplos caminhos para múltiplas condições.

Estrutura if / elif / else

Quando temos mais de dois cenários:


media = float(input("Média final: "))

if media >= 7:
    print("Aprovado")
elif media >= 5:
    print("Recuperação")
else:
    print("Reprovado")
O Python testa de cima para baixo. Assim que encontra uma condição verdadeira, executa aquele bloco e ignora o resto.

Fluxo de Decisão — Notas

Situação Condição Saída
Aprovado media >= 7 "Aprovado"
Recuperação media >= 5 (e < 7) "Recuperação"
Reprovado Nenhuma das anteriores "Reprovado"
No elif não precisa colocar media < 7 porque se chegou ali, o primeiro if já foi falso!

Micro Fixação

Qual a saída?


temperatura = 35

if temperatura > 30:
    print("Muito quente 🔥")
elif temperatura > 20:
    print("Agradável 😊")
else:
    print("Frio 🥶")
a) Muito quente 🔥
b) Agradável 😊
c) Frio 🥶
d) Muito quente 🔥 e Agradável 😊
Resposta: a) — 35 > 30 é verdadeiro, então executa só o primeiro bloco. O elif nem é testado!

Parte 4

Condições Compostas

Combinando condições com and, or e not.

Operadores Lógicos

and — todas as condições precisam ser verdadeiras.
or — basta uma condição ser verdadeira.
not — inverte o valor lógico. 

idade = 20
tem_cnh = True

if idade >= 18 and tem_cnh:
    print("Pode dirigir 🚗")
else:
    print("Não pode dirigir")

Tabela Verdade — and / or

ABA and B

TrueFalseFalse
FalseTrueFalse
FalseFalseFalse
ABA or B
TrueTrueTrue ✅
TrueFalseTrue ✅
FalseTrueTrue ✅
FalseFalseFalse

Parte 5

Condicionais Encadeadas

Um if dentro de outro if.

Condicional Encadeada

Podemos colocar um if dentro de outro para refinar a decisão:


usuario = input("Usuário: ")
senha = input("Senha: ")

if usuario == "admin":
    if senha == "python123":
        print("Acesso permitido ✅")
    else:
        print("Senha incorreta ❌")
else:
    print("Usuário inválido ❌")
Cada nível de if precisa de mais indentação. Cuidado para não se perder!

Micro Fixação

Encontre os 2 erros


idade = int(input("Idade: "))

if idade = 18:
    print("Tem 18 anos")
elif idade > 18
    print("Maior de 18")
else:
    print("Menor de 18")
Erros: Linha 3: usar = em vez de ==. Linha 5: falta o : no final do elif.

Erros Comuns

  • Esquecer o : após if, elif ou else.
  • Indentação errada — o bloco não pertence ao if.
  • Usar = (atribuição) em vez de == (comparação).
  • Comparar texto sem padronizar maiúsculas/minúsculas. 

# ❌ ERRADO
if nota = 7
    print("ok")

# ✅ CORRETO
if nota == 7:
    print("ok")
