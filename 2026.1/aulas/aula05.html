Estrutura if

O Que é Condicional?

Na vida real, tomamos decisões o tempo todo:

🌧️
Se estiver chovendo, levo o guarda-chuva.
Senão, saio sem ele.
📝
Se a nota for ≥ 7, o aluno está aprovado.
Senão, está reprovado.
Em Python, fazemos isso com if, elif e else.

Fluxo de Decisão

O programa avalia uma condição e escolhe um caminho:

Em Python, o bloco de código é definido pela indentação (4 espaços).

Estrutura if

O if executa um bloco somente se a condição for verdadeira:


idade = int(input("Digite sua idade: "))

if idade >= 18:
    print("Maior de idade")
Se a condição for falsa, nada acontece — o programa simplesmente continua.

Parte 2

Estrutura if / else

Dois caminhos: um para verdadeiro, outro para falso.

Estrutura if / else

Quando queremos que algo aconteça nos dois casos:


nota = float(input("Digite a nota: "))

if nota >= 7:
    print("Aprovado")
else:
    print("Reprovado")
O else pega todos os outros casos que o if não pegou.

Parte 3

Estrutura if / elif / else

Múltiplos caminhos para múltiplas condições.

Estrutura if / elif / else

Quando temos mais de dois cenários:


media = float(input("Média final: "))

if media >= 7:
    print("Aprovado")
elif media >= 5:
    print("Recuperação")
else:
    print("Reprovado")
O Python testa de cima para baixo. Assim que encontra uma condição verdadeira, executa aquele bloco e ignora o resto.

Fluxo de Decisão — Notas

Situação Condição Saída
Aprovado media >= 7 "Aprovado"
Recuperação media >= 5 (e < 7) "Recuperação"
Reprovado Nenhuma das anteriores "Reprovado"
No elif não precisa colocar media < 7 porque se chegou ali, o primeiro if já foi falso!
Prática

Atividades de Decisão

Vamos praticar decisões com if, if/else e if/elif/else.

Atividade de Decisão 1 de 3

Classificação por Idade

Leia a idade de uma pessoa e classifique:

  • Menor de idade se idade < 18
  • Adulto se idade entre 18 e 59
  • Idoso se idade >= 60

Exiba apenas a classificação final.

Dica: use if / elif / else para cobrir os três cenários.

Resposta — Atividade de Decisão 1

Solução


idade = int(input("Digite a idade: "))

if idade < 18:
    print("Menor de idade")
elif idade < 60:
    print("Adulto")
else:
    print("Idoso")
Pontos-chave: a ordem das condições importa. Ao entrar em um bloco verdadeiro, os próximos não são avaliados.

Atividade de Decisão 2 de 3

Desconto por Valor da Compra

Leia o valor de uma compra.

Aplique as regras:

  • Se valor for maior ou igual a 200, desconto de 10%
  • Caso contrário, desconto de 5%

Exiba o valor final com desconto.

Dica: guarde o percentual em uma variável e reutilize no cálculo.

Resposta — Atividade de Decisão 2

Solução


valor = float(input("Valor da compra: R$ "))

if valor >= 200:
    desconto = 0.10
else:
    desconto = 0.05

total = valor * (1 - desconto)

print(f"Desconto aplicado: {desconto * 100:.0f}%")
print(f"Valor final: R$ {total:.2f}")
Pontos-chave: a decisão define o percentual. O cálculo final usa o fator (1 - desconto).

Atividade de Decisão 3 de 3

Situação Final do Aluno

Leia a média final e exiba a situação:

  • Aprovado se média >= 7
  • Recuperação se média >= 5 e < 7
  • Reprovado caso contrário

Dica: no elif, não precisa repetir tudo que o primeiro if já descartou.

Resposta — Atividade de Decisão 3

Solução


media = float(input("Média final: "))

if media >= 7:
    print("Aprovado")
elif media >= 5:
    print("Recuperação")
else:
    print("Reprovado")
Pontos-chave: a leitura é sequencial: primeiro testa >= 7, depois >= 5, e o restante cai no else.

Resumo da Aula

if

Executa bloco quando a condição é verdadeira

else

Executa bloco alternativo (caso falso)

elif

Condições intermediárias extras

Não esqueça: dois pontos : no final e indentação no bloco!
