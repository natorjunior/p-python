Dicionários em Python
O que é um dicionário?

Um dicionário é uma coleção de pares chave: valor que permite acesso rápido às informações.

Exemplo 1: Dicionário Simples

planeta = {"nome": "Marte"}
Exemplo 2: Dicionário com Vários Dados

planeta = {
    "nome": "Marte",
    "diametro": 6779,
    "luas": 2
}
Acessando Elementos

print(planeta["nome"])      # Exibe o nome do planeta
print(planeta["diametro"])  # Exibe o diâmetro
Adicionando e Atualizando Dados

# Adicionando nova informação
planeta["distancia"] = 225e6  # distância em km

# Atualizando valor existente
planeta["diametro"] = 6792

print(planeta)
Removendo Elementos

del planeta["luas"]
print(planeta)
Iterando sobre o Dicionário

for chave, valor in planeta.items():
    print(f"{chave}: {valor}")

Tuplas em Python
O que é uma tupla?

Uma tupla é uma sequência imutável de elementos, utilizada para armazenar valores que não serão alterados.

Exemplo 1: Tupla Simples

      planeta = ("Marte",)
Exemplo 2: Tupla com Múltiplos Elementos

      planetas = ("Marte", "Terra", "Júpiter")
Acessando Elementos

      print(planetas[0])  # Exibe "Marte"
      print(planetas[1])  # Exibe "Terra"
Slicing em Tuplas

      print(planetas[1:])  # Exibe ("Terra", "Júpiter")
Iterando sobre Tuplas

      for p in planetas:
          print(p)
