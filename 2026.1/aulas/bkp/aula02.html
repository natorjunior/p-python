Funções no Python

Definição de Função

Funções são blocos de código reutilizáveis que realizam tarefas específicas.

def saudacao():
    print("Olá, bem-vindo ao Python!")

saudacao()  # Chama a função

Funções com Parâmetros

Podemos passar valores para funções para torná-las mais flexíveis.

def somar(a, b):
    return a + b

resultado = somar(5, 3)
print(resultado)  # Saída: 8

Funções com Retorno

Podemos retornar valores de uma função usando return.

def quadrado(n):
    return n ** 2

print(quadrado(4))  # Saída: 16

Funções Lambda

Funções anônimas podem ser criadas com lambda.

dobro = lambda x: x * 2
print(dobro(5))  # Saída: 10

Funções com Parâmetros Opcionais

Podemos definir valores padrão para os parâmetros.

def saudacao(nome="Visitante"):
    print(f"Olá, {nome}!")

saudacao()  # Olá, Visitante!
saudacao("João")  # Olá, João!

Funções Recursivas

Funções podem chamar a si mesmas para resolver problemas recursivos.

def fatorial(n):
    if n == 0:
        return 1
    return n * fatorial(n - 1)

print(fatorial(5))  # Saída: 120

Listas

Estruturas mutáveis que armazenam sequências ordenadas e oferecem diversos métodos para manipulação.

frutas = ["maçã", "banana", "laranja"]
frutas.append("uva")
print(frutas)




Funções Internas de Listas no Python

Método append()

Adiciona um item ao final da lista.

  lista = [1, 2, 3]
lista.append(4)  # [1, 2, 3, 4]

Método insert()

Insere um item em uma posição específica.

  lista.insert(1, 5)  # [1, 5, 2, 3, 4]

Método pop()

Remove e retorna o último item (ou o índice especificado).

  item = lista.pop()  # Remove e retorna o último elemento
print(item, lista)  # Exibe o item removido e a lista resultante

Método del()

Remove um item pelo índice ou a lista inteira.

  del lista[0]  # Remove o primeiro elemento
print(lista)  # Exibe a lista após a remoção

Método remove()

Remove um item específico pelo seu valor.

  lista.remove(2)  # Remove o número 2 da lista
print(lista)  # Exibe a lista atualizada

Método sort()

Ordena a lista em ordem crescente.

  lista.extend([7, 1])  # Adiciona novos elementos
lista.sort()  # Ordena em ordem crescente
print(lista)  # Exibe a lista ordenada

Método reverse()

Inverte a ordem dos elementos da lista.

  lista.reverse()  # Inverte a ordem da lista
print(lista)  # Exibe a lista invertida

Dicionários no Python

O que são Dicionários?

Dicionários são estruturas de dados que armazenam pares chave: valor, permitindo acesso eficiente aos dados.

        dados = {"nome": "Alice", "idade": 25, "cidade": "São Paulo"}
print(dados["nome"])  # Saída: Alice

Acessando Valores

Podemos acessar valores diretamente pela chave.

        print(dados.get("idade"))  # Saída: 25
print(dados.get("email", "Não informado"))  # Saída: Não informado

Adicionando e Atualizando Valores

Podemos adicionar novos pares ou atualizar valores existentes.

        dados["profissao"] = "Engenheira"
dados["idade"] = 26
print(dados)

Removendo Itens

Podemos remover elementos com del ou pop().

        del dados["cidade"]
idade = dados.pop("idade")
print(dados)  # {'nome': 'Alice', 'profissao': 'Engenheira'}

Iterando sobre um Dicionário

Podemos percorrer as chaves, valores ou ambos.

        for chave, valor in dados.items():
    print(f"{chave}: {valor}")

Dicionários Aninhados

Dicionários podem conter outros dicionários.

usuarios = {
  "alice": {"idade": 25, "cidade": "São Paulo"},
  "bob": {"idade": 30, "cidade": "Rio de Janeiro"}
}
print(usuarios["bob"]["cidade"])  # Saída: Rio de Janeiro
