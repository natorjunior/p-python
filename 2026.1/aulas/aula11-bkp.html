Conceito

Por Que Usar Funcoes?

Funcoes ajudam a dividir o problema em partes pequenas e reutilizaveis.

Repeticao vs Reuso

Sem funcao (repetitivo)


n1 = 8
n2 = 6
media = (n1 + n2) / 2
print(media)

n1 = 9
n2 = 7
media = (n1 + n2) / 2
print(media)

Com funcao (reutilizavel)


def calcular_media(a, b):
    return (a + b) / 2

print(calcular_media(8, 6))
print(calcular_media(9, 7))

Sintaxe Basica


def saudacao(nome):
    print("Ola,", nome)

saudacao("Ana")
saudacao("Bruno")
  • def inicia a definicao da funcao.
  • nome e o parametro de entrada.
  • O bloco indentado e o corpo da funcao.

Parametros


def potencia(base, expoente=2):
    return base ** expoente

print(potencia(5))      # 25
print(potencia(3, 3))   # 27
  • Podemos definir valor padrao para parametros.
  • Isso deixa a chamada da funcao mais flexivel.

Retorno com return


def dobro(numero):
    resultado = numero * 2
    return resultado

x = dobro(7)
print(x)  # 14

Funcoes com return devolvem valor para ser usado em outra parte do programa.

Atividade F1 — Funcao de Media

Pratica inicial

Crie uma funcao chamada media3(a, b, c).

A funcao deve retornar a media de tres numeros.

Depois, leia 3 notas do usuario e exiba a media final usando a funcao.

Tempo sugerido: 06:00

Solucao — Atividade F1

Codigo


def media3(a, b, c):
    return (a + b + c) / 3

n1 = float(input("Nota 1: "))
n2 = float(input("Nota 2: "))
n3 = float(input("Nota 3: "))

m = media3(n1, n2, n3)
print("Media:", m)
Repare que a formula da media ficou isolada e pode ser reutilizada em qualquer programa.
Escopo

Variavel Local x Global

Escopo define onde cada variavel pode ser acessada.

Exemplo de Escopo


msg = "fora"

def teste():
    msg = "dentro"
    print(msg)  # dentro

teste()
print(msg)      # fora
  • A variavel criada dentro da funcao e local.
  • Ela nao altera a variavel global com mesmo nome.

Sem retorno x Com retorno

Tipo Quando usar Exemplo
Sem return Quando a funcao so executa uma acao def menu(): print("1-Entrar")
Com return Quando queremos reutilizar o resultado def soma(a,b): return a+b

Atividade F2 — Classificar Nota

Funcao com if/elif/else

Crie a funcao situacao(nota) que retorne:

"aprovado" se nota >= 7

"recuperacao" se nota >= 5 e < 7

"reprovado" se nota < 5

Depois teste a funcao com 4 notas diferentes.

Tempo sugerido: 07:00

Solucao — Atividade F2

Codigo


def situacao(nota):
    if nota >= 7:
        return "aprovado"
    elif nota >= 5:
        return "recuperacao"
    else:
        return "reprovado"

for valor in [8.5, 6.2, 4.0, 7.0]:
    print(valor, "->", situacao(valor))
A regra de negocio ficou encapsulada em uma funcao unica e facil de manter.

Boas Praticas

  • Use nomes claros para funcoes: calcular_media(), situacao().
  • Cada funcao deve ter uma responsabilidade principal.
  • Prefira retorno de valor em vez de imprimir dentro da funcao (quando fizer sentido).
  • Teste a funcao com entradas diferentes para validar comportamento.
