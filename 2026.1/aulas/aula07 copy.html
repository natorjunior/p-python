Laços de Repetição em Python

Começando com while e avançando para for com exemplos bem didáticos.

whileforbreakcontinuerange()

Prof. Msc. Nator Junior Carvalho da Costa

Objetivos da Aula

  • Entender a ideia de repetir instruções em programação.
  • Aprender o while: repetir enquanto uma condição for verdadeira.
  • Aprender o for: repetir uma quantidade conhecida de vezes.
  • Conhecer break e continue para controlar o fluxo.
  • Escolher a estrutura mais adequada em cada situação.
Meta: sair escrevendo laços simples sem cair em laço infinito.

Quando usar repetição?

Situações típicas

  • Tabuada
  • Contagem regressiva
  • Leitura até critério de parada
  • Processar cada item de uma lista

Sem laço

Código repetido aumenta chance de erro e dificulta manutenção.

Laços tornam o algoritmo mais claro e escalável.

Motivação: tabuada sem laço


num = int(input("Digite um número: "))
print(num, "x 1 =", num * 1)
print(num, "x 2 =", num * 2)
print(num, "x 3 =", num * 3)
# ... até x 9
A estrutura é repetitiva e muda apenas o fator multiplicador.
Parte 1

Laço while

Repete um bloco enquanto uma condição for verdadeira.

Sintaxe do while


while condicao:
    # bloco de repetição
    # atualização da variável de iteração

Leitura: "enquanto a condição for verdadeira, execute este bloco."

Ponto-chave: a variável de iteração precisa mudar a cada repetição.

Fluxo do while

Início
➡
Condição?
✅
Executa bloco
↩
Testa de novo
❌
Fim

Exemplo: tabuada com while


num = int(input("Digite um número: "))
n = 1

while n < 10:
    print(f"{num} x {n} = {num * n}")
    n = n + 1
n é a variável de iteração: inicia em 1 e avança até 9.

O que acontece aqui?

n = 0, então n > 0 é falsa desde o início.

O bloco do while nunca executa.

O programa pula direto para print('Secar!').

Secar!

E agora?

n = 5, então n > 0 é verdadeira.

O laço executa... mas n nunca muda!

A condição continua verdadeira para sempre.

→ Laço infinito. O programa trava.

Ensaboar Enxaguar Ensaboar Enxaguar ... (nunca termina)

E aqui?

Agora n é decrementado a cada volta.

O laço roda enquanto n > 0: valores 5, 4, 3, 2, 1.

Quando n = 0, a condição falha — o laço termina.

5 4 3 2 1 Fim!

Atividade 1 — while

Soma de 1 até N

Leia n e calcule a soma de 1 até n usando while.

Exemplo: n = 5 → resultado 15.

Use uma variável acumuladora e outra de iteração.

Resposta — Atividade 1

Solução com while


n = int(input("Digite n: "))
contador = 1
soma = 0

while contador <= n:
    soma += contador
    contador += 1

print(f"Soma de 1 até {n}: {soma}")
Fluxo: inicializar → testar condição → acumular → atualizar → repetir.
Parte 2

Laço for

Mais indicado quando a quantidade de repetições é conhecida.

Sintaxe do for com range()


for variavel in range(inicio, fim, passo):
    # bloco de repetição
  • inicio: opcional, padrão 0.
  • fim: limite superior (não incluído).
  • passo: opcional, padrão 1.

Exemplos de range()


for i in range(5):
    print(i)        # 0, 1, 2, 3, 4

for i in range(1, 6):
    print(i)        # 1, 2, 3, 4, 5

for i in range(10, 0, -2):
    print(i)        # 10, 8, 6, 4, 2

Exemplo: percorrendo caracteres


nome = "python"
for letra in nome:
    print(letra)

Atividade 2 — for

Tabuada com for

Leia um número inteiro e exiba a tabuada de 1 até 10.

Formato sugerido: 7 x 3 = 21.

Resposta — Atividade 2

Solução com for


numero = int(input("Digite um número: "))
for i in range(1, 11):
    print(f"{numero} x {i} = {numero * i}")
Quando a repetição é fixa, o for deixa o código mais direto.
Parte 3

break e continue

Controle fino de fluxo dentro dos laços.

break: encerra o laço


while True:
    linha = input("> ")
    if linha == "fim":
        break
    print(linha)
print("Agora é o fim!")
break interrompe o laço corrente e segue para a próxima instrução após ele.

continue: pula a iteração atual


while True:
    linha = input("> ")
    if linha.startswith("#"):
        continue
    if linha == "fim":
        break
    print(linha)
print("Agora é o fim!")
continue encerra só a iteração atual e retorna ao teste da condição.

Escolha da estrutura

Cenário Estrutura indicada Exemplo
Quantidade desconhecida de repetições while Validar senha
Quantidade fixa de repetições for Tabuada de 1 a 10
Interromper o laço imediatamente break Sair ao digitar "fim"
Ignorar apenas uma iteração continue Ignorar linhas de comentário

Desafio rápido (em dupla)

Contador de linhas válidas

Crie um programa que:

1. Leia entradas em loop com while True.

2. Ignore linhas que começam com # usando continue.

3. Finalize ao receber fim com break.

4. Mostre no final quantas linhas válidas foram processadas.
